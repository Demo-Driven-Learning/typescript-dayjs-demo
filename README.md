TypeScript tsconfig.json Demo
============================

`tsconfig.json`是typescript项目的说明文件，指定`tsc`所需要的各个参数。

```
npm install
npx tsc
```

You will see `hello.js` and `hello.js.map` generated.

```
node hello.js
```

It will print `Hello, TypeScript!`

或者使用node-ts：

```
npx ts-node hello.ts
```
